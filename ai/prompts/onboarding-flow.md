# Prompt Template: Onboarding Flow

Create an onboarding flow for a new user or admin.

## Required Decisions

- What is the next best action?
- Which tasks are already complete?
- Which states require reassurance or warning?
- What can be deferred?

## Guardrails

Keep task labels action-oriented. Represent status with text and approved badges, not color alone. Avoid inventing one-off components for checklists, progress, or summaries.
