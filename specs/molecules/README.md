# Molecules

Molecules combine atoms into reusable product UI patterns.

## Approved Molecules

- `alert.md`
- `card.md`
- `table.md`
- `tabs.md`

## AI Rules

- Use molecules for repeated product patterns such as status feedback, grouped summaries, comparable data, and sibling-view switching.
- Do not nest cards inside cards.
- Do not use molecules as page chrome unless an organism spec allows it.
