# Organisms

Organisms are approved compositions used by the showcase generator and review UI.

## Approved Organisms

- `generated-screen.md`
- `compliance-review.md`
- `prompt-composer.md`

## AI Rules

- Organisms should compose approved atoms and molecules.
- Organisms may define layout, hierarchy, and behavior but should not introduce new visual primitives.
- If a prompt needs a pattern not represented here, report a component or organism gap.
