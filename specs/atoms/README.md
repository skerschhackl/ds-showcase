# Atoms

Atoms are the smallest approved UI building blocks available to generated screens.

## Approved Atoms

- `button.md`
- `input.md`
- `select.md`
- `field.md`
- `textarea.md`
- `badge.md`
- `spinner.md`

## AI Rules

- Use atoms directly for simple forms, actions, statuses, and loading states.
- Combine atoms inside molecules and organisms only when the composition remains clear.
- Do not invent atom variants outside each component contract.
